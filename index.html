Modern debate is broken:

We're drowning in arguments and evidence — but everything’s repeated, disorganized, and disconnected.

  • We argue in circles — same points, no progress.
  • No one links claims to shared evidence, so facts get lost in the noise.
  • Beliefs get pushed without measuring real-world costs, benefits, or logic.

What’s the fix?

A platform that automates conflict resolution, ranks evidence, and structures cost-benefit analysis. Let’s stop spinning our wheels and actually get somewhere.

Key Features

  1. Evidence-Driven Framework
    • One page per topic: pros on one side, cons on the other.
    • ReasonRank: like Google’s PageRank, but for arguments.
    • Dynamic Scoring: updates as users add new evidence or counterpoints.
  2. Analysis Tools for Smarter Decisions
    • Automated Cost-Benefit Analysis
    • Likelihood Scores
    • Maslow Mapping
  3. Built-In Conflict Resolution
    • Interest Mapping
    • Motivation Insight
    • Consensus Builders
